# Chemical Compound Image Analyzer

A local-hosted and Streamlit-based application for analyzing chemical compound images using
Ollama-powered multimodal and language models (LLaVA and Gemma).

## Features
- Upload chemical compound images
- Visual analysis using LLaVA
- Academic refinement using Gemma
- Streamlit-based UI

## Tech Stack
- Python
- Streamlit
- Ollama (LLaVA + Gemma)

## Usage
Run locally:
```bash
streamlit run app.py
